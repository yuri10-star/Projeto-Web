# Lendas das Copas do Mundo

Projeto desenvolvido para a disciplina de **Programação Web I**.

## Sobre o projeto

Este projeto tem como objetivo desenvolver uma página web responsiva com o tema **Lendas das Copas do Mundo**, apresentando alguns dos maiores jogadores da história do torneio.

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Git
- GitHub

## responsividade implementada e detalhes

- implementei a responsividade fazendo testes em quatro dispositivos diferentes do Chrome devtools, iphone SE, iphone 14 Pro, Ipad Air e o proprio desktop. Todos funcionaram corretamente.
- para dispositivos menores algumas seções se adaptam para atender as necessidades da dimensão da tela.
- a navbar é substituída por um menu hambúrguer para que fique acessivel ao usuario que possui dispositivos com telas menores
- a section inicial tem a imagem de background reposicionada para centralizar a figura principal que é o pelé.
- as sections de jogadores e curiosidades sairam de cards alinhados em containers para um carrossel que contem o botão para passar as imagens mas tambem passa automaticamente de acordo com o tempo
- foi adicionado um botão para voltar ao topo da pagina, esse botão e as outras funcionalidades funcionam em dispositivos com display abaixo de 993px, onde o tamanho da tela nao ficaria tao agradavel.
- a tabela de estatisticas diminui de acordo com a largura da tela, em dispositivos mobile exibe apenas 3 informações, para que possa caber na tela
